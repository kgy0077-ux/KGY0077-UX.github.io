사실조사

행정 관련 현지 사실 확인·조사 및 공적 확인 서류 작성 전문

주요 업무

🏠

부동산 현황 조사

토지·건물의 현재 상태, 점유 현황, 이용 상태 확인

👤

거주 사실 확인

실제 거주 여부, 주소지 일치 여부 현지 확인

🏪

영업 사실 확인

사업장 운영 실태, 영업 여부, 종업원 현황 확인

📸

현황 사진·영상 기록

행정·법률 목적의 현장 사진 및 영상 기록 작성

📄

사실확인서 작성

행정사 명의 사실확인서 작성·발급 (법적 효력)

🌾

농지·임야 이용 확인

농지 실제 경작 여부, 임야 현황 사실 확인

⚙️

시설·설비 현황 조사

인허가 시설 기준 충족 여부 현장 확인 조사

📊

행정조사 보고서 작성

조사 결과를 체계적으로 정리한 공식 보고서 작성

사실조사가 필요한 경우

  • 소송·행정심판 증거 자료가 필요한 경우
  • 인허가 신청 시 현황 확인 서류가 필요한 경우
  • 부동산 거래 또는 상속·증여 시 현황 파악이 필요한 경우
  • 보험·손해배상 청구를 위한 현장 증빙이 필요한 경우
  • 해외 거주자의 국내 재산·거주 실태 확인이 필요한 경우
  • 공공기관 제출용 사실확인서가 필요한 경우
  • 건물·토지의 무단점유·불법사용 확인이 필요한 경우

처리 절차

  1. 조사 목적 및 범위 확정

    의뢰인과 조사 목적, 대상, 범위, 방법 협의

  2. 사전 자료 검토

    등기부등본, 토지대장, 건축물대장 등 공부 검토

  3. 현장 조사 실시

    현지 방문, 사진·영상 촬영, 관계인 면담

  4. 조사 보고서 작성

    조사 결과 정리, 사실확인서 또는 조사 보고서 작성

  5. 서류 납부 및 사후 지원

    완성된 서류 전달, 필요 시 추가 확인·보완
